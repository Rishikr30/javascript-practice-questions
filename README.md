# JavaScript Practice Questions (Basic to Advanced)  

Welcome to the **JavaScript Practice Questions** repository! This repository contains **50 JavaScript coding questions** along with their solutions, ranging from **basic** to **advanced** levels. Whether you're a beginner learning JavaScript or an experienced developer brushing up on your skills, these exercises will help strengthen your problem-solving abilities.  

## 📂 Repository Structure  

- `questions/` → Contains JavaScript practice questions in separate files  
- `solutions/` → Contains corresponding solutions with explanations  

## 🔥 Topics Covered  

✔️ Variables & Data Types  
✔️ Functions & Scope  
✔️ Loops & Conditionals  
✔️ Arrays & Objects  
✔️ Higher-Order Functions  
✔️ ES6+ Features  
✔️ Asynchronous JavaScript (Promises, async/await)  
✔️ Error Handling  
✔️ Closures & Hoisting  
✔️ DOM Manipulation  

## 🚀 How to Use  

1. Clone the repository:  
   ```bash
   git clone https://github.com/Rishikr30/javascript-practice-questions.git
   cd javascript-practice-questions
   ```  
2. Open the `questions/` folder and try solving the problems on your own.  
3. Compare your solution with the `solutions/` folder for explanations.  
4. Run the solutions using Node.js or in the browser console:  
   ```bash
   node solutions/question1.js
   ```
